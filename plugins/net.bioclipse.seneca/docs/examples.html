SENECA Examples

Four examples are provided for the SENECA Bioclipse plugin.

Installation

The example files are installed via the install functionality in the main menu.

Example 1

This example shows CASE based on a 1D 13C NMR spectrum for a small molecule (C5H10O2) using the deterministic structure generator.

InChI=1/C5H10O2/c1-3-5(6)7-4-2/h3-4H2,1-2H3

Example 2

The second example shows CASE for a larger structure (C10H16) using the stochastic structure generator and again 1D 13C NMR input. This generator, however, needs the be made aware of the number of hydrogens attached to the carbons, as extracted from a DEPT experiment. This data is available in the example files.

InChI=1/C10H16/c1-7-4-5-8-6-9(7)10(8,2)3/h4,8-9H,5-6H2,1-3H3
