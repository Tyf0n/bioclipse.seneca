Writing a Scoring Function

Scoring functions (also called judges) are contributed via the t.bioclipse.seneca.SenecaJudge extension. An example entry could look like this:

The schema for the extension point can found here. The judge class must implement the net.bioclipse.seneca.judge.IJudge interface. In most cases, the abstract class net.bioclipse.seneca.judge.Judge can be extended. The class should then implement the empty constructor and the methods createJudge, evaluate, getDescription, checkJudge and setData (see JavaDoc for explanation). The judge should then show up in the editor and can be used.

Important considerations:

  • The evaluate method needs to return a score starting with 0 (very bad) to calcMaxScore() (very good).
